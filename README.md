# QWON

QWON is a local-first mobile AI runtime for iPhone.

**Website:** [https://qwon.ai](https://qwon.ai)

## Focus

- On-device intelligence
- Semantic routing
- Multimodal processing
- Context compression
- Cloud LLM orchestration

## Repository Layout

- `app/ios/`: iOS app code
- `runtime/`: orchestration, memory, multimodal, LLM, and RAG runtime modules
- `models/`: local model assets, prompts, and benchmarks
- `tools/`: scripts, converters, benchmarking, and profiling helpers
- `backend/`: relay, telemetry, auth, and analytics services
- `docs/`: requirements, product, API, and research notes
- `website/`: landing page and media assets
- `design/`: UI/UX, logo, icon, and mockup sources
- `experiments/`: prototype and evaluation work
- `test/`: datasets, prompts, scenarios, and benchmarks
