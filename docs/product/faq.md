# FAQ

## QWON とは？

QWON は、iPhone 向けのローカルファースト AI ランタイムです。単なるチャットアプリではなく、ローカル推論、ルーティング、コンテキスト圧縮、メモリ、マルチモーダル入力を統合した実行基盤を目指しています。

## なぜチャットアプリではないのですか？

価値の中心が UI ではなく runtime にあるからです。QWON は、入力を理解し、文脈を圧縮し、必要な記憶を呼び出し、最適なモデルへ自動で振り分ける知能レイヤを作ろうとしています。

## なぜ iPhone を中心にするのですか？

iPhone は常時携帯され、高品質なカメラ、マイク、各種センサー、Neural Engine を持っています。QWON はこれを単なる表示端末ではなく、常時利用可能なセンサープラットフォームとして使います。

## すべてローカルで動きますか？

いいえ。基本方針はローカルファーストですが、高度な推論や長文解析、高品質生成などは必要に応じてクラウドへエスカレーションします。ただし、その前にローカルで分類と圧縮を行います。

## なぜローカル LLM を使うのですか？

ローカル LLM は、会話そのものだけでなく、intent 解析、ルーティング、圧縮、メモリ制御の中核に使えます。これにより、遅延、コスト、プライバシーの面で有利になります。

## どのクラウドモデルを使いますか？

初期対象は OpenAI と Anthropic です。Gemini や LAN 内モデルはロードマップ上の拡張候補です。

## API キーはどう扱いますか？

ユーザー自身が設定し、iOS Keychain で保管する想定です。キーをコードに埋め込むことはしません。

## オフラインでも使えますか？

MVP では、基本会話、OCR、音声入力、ローカル検索、ローカルメモリをオフラインで使えることを目標にします。高度推論はオンライン依存になります。

## QWON の差別化は何ですか？

ローカル中心、クラウド依存最小化、セマンティックルーティング、文脈圧縮、マルチモーダル前処理、パーソナルメモリの組み合わせです。

## 誰向けですか？

最初は開発者、研究者、AI パワーユーザー、クリエイターなど、要求水準が高く API やモデル活用に慣れた層を対象にしています。

## MVP で何が入りますか？

チャット UI、ローカル推論経路、OpenAI / Anthropic 接続、自動モデル振り分け、コンテキスト圧縮、OCR、基本 Vision、API キー管理が中心です。

## 何を最初からやらないのですか？

クラウド前提の設計、ライトユーザー向けの広範な簡略化、常時重いバックグラウンド処理、ソーシャル機能、汎用的なチャットアプリ化は優先しません。
