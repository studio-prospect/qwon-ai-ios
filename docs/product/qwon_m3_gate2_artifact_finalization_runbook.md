## Required artifact record

Release Engineering should produce one written record with these fields.

| Field | Requirement |
| --- | --- |
| **Artifact owner** | Release Engineering operational owner and Product decision owner |
| **Source artifact** | The upstream or internal artifact used to create the hosted object |
| **Hosted object identity** | QWON-owned bucket/object key plus version ID, etag, immutable revision, or equivalent |
| **Product URL class** | QWON-owned product path; avoid third-party Hugging Face URLs as tester/product-facing endpoint unless separately approved |
| **On-device filename** | `prexus-local-mvp.gguf` — preserved |
| **Exact byte size** | A single integer byte count for the final hosted object |
| **SHA-256** | Full lowercase hex SHA-256 for the final hosted object |
| **Generated by** | Command(s), machine, date, and operator |
| **Verification repeat** | At least one repeat hash/size check against the same hosted object or local staged object |
| **Storage location for evidence** | Ops folder or release record path outside git |

Do **not** commit the GGUF file, downloaded blob, archive, or raw logs. Only the final metadata record belongs in docs.

---

## Recommended local evidence commands

Run these only after the object intended for QWON hosting is staged locally.

```bash
shasum -a 256 models/prexus-local-mvp.gguf
stat -f%z models/prexus-local-mvp.gguf
```

If the final object is downloaded back from QWON hosting for verification, run the same commands on that downloaded file and confirm the values match.

| Check | Expected output shape |
| --- | --- |
| **SHA-256** | `64_hex_chars  path/to/file.gguf` |
| **Byte size** | Integer bytes only |
| **Mismatch** | Treat as artifact not finalized; do not answer Q-A-06/Q-A-07 |

## Non-approvals

This runbook does **not** approve:

- M3 in-app download implementation
- Swift/network/storage schema changes
- Build `4`, TestFlight upload, tag, or version bump
- Gate 2 Ready sign-off
- Replacing `prexus-local-mvp.gguf`
- Silent overwrite of existing USB-placed GGUF files
