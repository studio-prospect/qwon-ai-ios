# PREXUS 要件定義書 v0.1

## 1. プロジェクト概要

### 1.1 プロジェクト名
PREXUS

### 1.2 コンセプト
PREXUS は、iPhone 上で動作するローカルLLMを中核とした AI ハーネス / AI ランタイムである。

単なるチャットクライアントではなく、

- ローカル推論
- マルチモーダル入力
- セマンティックルーティング
- クラウドLLMオーケストレーション
- コンテキスト圧縮
- パーソナルメモリ

を統合した「携帯型AI実行基盤」を目指す。

PREXUS は、小型ローカルLLMを常時利用可能な知能層として動作させ、必要時のみ高性能クラウドLLMへ推論をエスカレーションする。

これにより、

- APIコスト削減
- レイテンシ低減
- オフライン動作
- プライバシー保護
- バッテリー効率改善

を実現する。

---

# 2. 開発目的

## 2.1 背景
現在の主要AIアプリは以下の課題を持つ。

- 全入力をクラウドへ送信する
- 巨大コンテキストを毎回投入する
- 高コスト
- 高レイテンシ
- プライバシー懸念
- ネットワーク依存

また、複数LLMを使い分ける際の UX が複雑であり、一般的ユーザーにはモデル選択が困難である。

PREXUS は、ローカルLLMをオーケストレータとして活用し、用途ごとに最適な推論先を自動選択することで、この問題を解決する。

# 6. コアコンセプト

## 6.1 ローカルファースト
可能な限りオンデバイスで処理を行う。

### ローカル処理対象
- 会話
- 要約
- intent分類
- OCR
- 音声認識
- センサー解析
- embedding
- コンテキスト圧縮
- 軽量RAG

## 6.2 クラウドエスカレーション
高度推論が必要な場合のみクラウドLLMへ処理を委譲する。

### 対象例
- 大規模コード解析
- 深い推論
- 長文生成
- 高品質創作
- 高精度vision解析

## 6.3 セマンティックルーティング
入力内容を解析し、最適なモデルへ自動振り分けを行う。

例：

| タスク | 実行先 |
|---|---|
| 要約 | ローカル |
| 音声認識 | ローカル |
| OCR | ローカル |
| 数学 | Gemini系 |
| コード解析 | GPT系 |
| 創作 | Claude系 |
| 機密データ | ローカル限定 |
